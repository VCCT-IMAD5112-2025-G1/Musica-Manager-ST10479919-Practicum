
# Musica Manager
## ST10479919 Practicum

An android add that manages playlists

## Features

- Ability to add:
        Song Titles,
        Artists,
        Ratings,
        Comment on songs.
- Ability to find average rating of songs

## FAQ

#### App crashes on startup

Please make sure you are using the latest avalible version of android
